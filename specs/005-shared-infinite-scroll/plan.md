# Implementation Plan: Shared Infinite Scroll

**Branch**: `005-shared-infinite-scroll` | **Date**: 2026-04-04 | **Spec**: [spec.md](./spec.md)  
**Input**: Feature specification from `/specs/005-shared-infinite-scroll/spec.md`

## Summary

Extract the infinite scroll logic from the listings `LoadMore` component into a shared, generic `useInfiniteScroll` hook and `InfiniteScrollSentinel` component. The hook uses `react-intersection-observer` and `AbortController` to provide paginated loading with filter-reset support, concurrent-fetch prevention, and error recovery. The existing listings feed is refactored to use the shared hook with zero regression.
