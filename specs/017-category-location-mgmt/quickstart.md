# Quickstart: Category and Location Management

## Accessing the Management Dashboard

The management dashboard is accessible to administrators at:
`/[locale]/dashboard/management`

## Managing Categories

1. **View Categories**: Navigate to the "Categories" tab.
2. **Create Category**: Click the "Add Category" button. Fill in the English name, Arabic name, and optional description.
3. **Edit Category**: Click the "Edit" icon on a category row.
4. **Delete Category**: Click the "Delete" icon. Note that deletion is blocked if the category is used by active listings.

## Managing Locations

1. **View Locations**: Navigate to the "Locations" tab.
2. **Create Location**: Click the "Add Location" button. Fill in the names and optional sort order.
3. **Edit Location**: Click the "Edit" icon on a location row.
4. **Delete Location**: Click the "Delete" icon. Note that deletion is blocked if the location is used by active listings.

## Search and Filter

Use the search bar at the top of each table to filter by name. The list updates in real-time as you type.
